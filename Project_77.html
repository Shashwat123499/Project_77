How to set up visual studio code: You need to first download visual studio code by going to either the microsoft store or on the internet or a third party website if you want (like ninite.com, however, you should know what you are doing if you want to download stuff from third party websites)

After that, you need to let the setup run (if you have downloaded from the internet), or you are ready to go if you have downloaded from the microsoft store.

Then, open visual studio code, click on file (top left corner of your screen)=> open folder=> selec the folder you want your project to be in. After that, create a new file in the left hand side pane, name it and save the changes. Then, go to the "extensions" tab, search for "Live Server" extension and install it. That's it, you are ready to go!

What is console screen: Console screen is the screen that is hidden inside a webpage, that is, not visible to the average viewer and it is used to first check if everything (the values/operations etc.) that are present in our webpage are working correctly or not.
